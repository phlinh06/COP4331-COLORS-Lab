The COLORS application was a lab assignment meant to practice using LAMP (Linux, Apache, MySQL, PHP). The application itself is designed to allow users to log in and manage color data. This project demonstrates backend API development, MySQL database integration, and client-server interaction.

How to run application:
git clone https://github.com/YOUR_USERNAME/colors-lab.git 
cd colors-lab

Move files to Apache web root:
/var/www/html/

Or if using MAMP/XAMPP locally, move project into:
htdocs/

Create MySQL database
Create required tables
Update database credentials in PHP files

Open browser:
http://localhost/public/

Assumption and limitations:
No security (2-factor authentication)
